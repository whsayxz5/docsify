# 代理政策



**一、政策说明**

**机具价格：**99元/台

**激活返现：**130元（99元商城退款，30元税后APP日结）

**激活条件：**90天内贷记卡交易（不含借记卡/扫码）满1万元

**达标时间：**自终端入库之日起 180 天之内，未激活终端按 50 元/台补机具差价

**![image.png](../media/1596612279505610.png)**

**二、分润及返现发放**

**分润：**日结，系统后台提现

**返现：**日结，系统后台提现

**激活：**日结，系统后台提现
